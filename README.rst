===================
Tmux Session Helper
===================


.. image:: https://img.shields.io/travis/ibejohn818/tm.svg
        :target: https://travis-ci.org/ibejohn818/tm


Start, Kill & Resume Tmux Sessions


Features
--------
List sessions ( You can also add a new one )

`tm`

Add session

`tm add`

Kill sessions

`tm kill`
